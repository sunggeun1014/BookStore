## 프로젝트 소개
WHALE BOOKS는 온라인서점 사이트로 책을 판매하는 사이트 입니다.<br>
운영관리자가 사이트를 원활하게 운영할 수 있도록 admin페이지도 함께 개발 하였습니다.<br>
